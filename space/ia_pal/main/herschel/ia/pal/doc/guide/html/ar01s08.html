8. Product Cloning

It is possible to clone a complete context tree from one storage to another. The operation is simple: 

# Get Context from first storage
context=storage1.load("urn:123").product

# Save Context to second storage 
storage2.save(context)

The second step invokes the cloning operation.

The cloning operation recursively clones the context tree starting from the context in the argument storage.save() call (the 'head' context), downwards to the terminals (contexts with no children or leaf products).

It repeats the same operation for the older versions of the head context, but only that context. In other words, if there are child contexts that happen to have older versions, they are not cloned.
